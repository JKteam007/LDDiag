LDDiag Architecture

LDDiag.exe
  • Main executable
  • Responsible for UI and control
  • Loads other libaries for specific functions
LDDiag.Util.dll
  • Utilities Library
  • Contains utility functions specific to LDDiag such as:
    • Core detection
    • Console detection
    • Ivanti dll loading
    • Dev tools
ES.Crypto.dll
  • Encryption Library
  • Responsible for:
    • Encryption
    • Decryption
    • Hashing
    • Certificate operations
ES.Http.dll
  • Web Library
  • Responsible for:
    • HTTP/S communications
ES.Database.dll
  • Database Library
  • Responsible for:
    • SQL connections
    • ODBC connections
    • Query execution
